### Editor Display
- **Font Size**: Adjust the text size in your editor (Small → Medium → Large)
- **Font Family**: Choose from available monospace fonts for optimal code readability
- **Word Wrap**: Toggle line wrapping for better visibility of long lines

## Editor Behavior

### Auto Save
Enable automatic saving of your work:
- **On**: Changes are saved automatically every set timer.
- **Off**: Manual saving required (Ctrl+S / Cmd+S)

## Network Configuration

### Security Settings
- **Disable TLS Verification**: Toggle to bypass certificate validation (for testing environments only)

### Proxy Configuration
Set up proxy settings for network requests:

> [SCREENSHOT_PLACEHOLDER: Proxy configuration popup showing host, port, and authentication fields]

**Configure:**
- Proxy host and port
- Authentication credentials (if required)
- Bypass rules for specific domains

## Updates

### Early Access Program
- **Enable Early Access**: Get the latest features automatically as they're released
  - When enabled, Voiden will automatically download and install updates
  - Access cutting-edge features before general release
  - Help improve Voiden by providing feedback on new functionality

## CLI & Terminal

### Terminal Enhancement
- **Enable Nerd Fonts**: Toggle to support enhanced icon sets in the integrated terminal
  - Better visual representation of file types and status indicators
  - Improved terminal experience with icon-rich displays

## Keyboard Shortcuts

### Essential Shortcuts
Voiden provides comprehensive keyboard shortcuts for faster navigation and workflow:

[SCREENSHOT_PLACEHOLDER: Keyboard shortcuts panel showing categorized shortcuts]

**Common Shortcuts:**
- **File Operations**: Create, save, and manage void files

- **Editing**: Code completion, formatting, and text manipulation

Customize shortcuts to match your preferred development environment and workflow patterns.
